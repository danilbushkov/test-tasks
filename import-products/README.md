# Импорт товаров

Задание представлено [здесь](./test_request_laravel_php_v2.docx).

## Зависимости

- `docker`
- `docker compose`
- `make`

## Установка

Для сборки выполните:

```
make build
```

Для выполнения команды может потребоваться права суперпользователя.

## Запуск

```
make run
```

Если не выполнена миграция базы данных, то выполните:

```
make migrate
```

## Остановка

```
make stop
```
